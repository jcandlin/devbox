/etc/hosts config
-----------------

ensure new servers have correct /etc/host file settings in the FQDN format
i.e.
Master Server- 
10.1.28.156     brsuxdvoap01.williamhill-dev.local      brsuxdvoap01 #master
10.1.28.157     brsuxdvoap02.williamhill-dev.local      brsuxdvoap02 #app
10.1.28.158     brsuxdvoap03.williamhill-dev.local      brsuxdvoap03 #proxy
App Server - 
10.1.28.156     brsuxdvoap01.williamhill-dev.local brsuxdvoap01 #master
10.1.28.158     cbs-adapter.williamhill-dev.remote # proxy
Proxy Server - 
10.1.28.156     brsuxdvoap01.williamhill-dev.local brsuxdvoap01 #master
10.1.28.157     cbs-adapter-node1 #node names
10.1.28.158     cbs-adapter-node2 #node names
